本目录放的是初始化系统需要的文件和数据，db2在db210.1和9.1下验证通过，Oracle10g验证通过。

1. crebas_inf.sql、crebas_inf_his.sql、crebas_datacheck_inf.sql和crebas_inf_datacheck.sql
	适用于Informix数据库的创建脚本。注意：不适用于4.1版本，未更新4.1版本变化。
	
2. crebas_ora.sql、crebas_ora_his.sql、crebas_datacheck_ora.sql和crebas_ora_datacheck.sql
	适用于Oracle数据库的创建脚本。
	
3. crebas_bd2.sql、crebas_db2_his.sql、crebas_datacheck_db2.sql和crebas_db2_datacheck.sql
	适用于DB2数据库的创建脚本。
	
4. initMap.sql
	代码初始化脚本。
	
5. crebas_village.sql
	适用于村镇银行征信库,修改某些表结构，增加FINANCEID字段。
	
6. crebas_view.sql
	创建视图脚本。
